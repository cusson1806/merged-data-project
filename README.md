# merged-data-project
## Project Description

This project demonstrates the process of **merging and consolidating multiple datasets into a single, structured dataset using Microsoft Excel**. The objective was to integrate crime records from different sources into a clean and consistent format, making the data suitable for subsequent analysis and reporting.

The merged dataset contains information on financial years, reporting quarters, police forces, offence groups, offence subgroups, offence descriptions, offence codes, and the corresponding number of recorded offences. Throughout the project, emphasis was placed on maintaining data integrity, ensuring consistency across records, and producing a unified dataset that can serve as a reliable foundation for further analytics.

### Key Highlights

* Merged multiple datasets into a single, comprehensive dataset.
* Standardized data fields to ensure consistency across records.
* Preserved data accuracy and integrity during the merging process.
* Organized the final dataset into a clean, analysis-ready format.
* Validated records to minimize duplication and inconsistencies.
* Prepared the data for downstream reporting, visualization, and statistical analysis.
